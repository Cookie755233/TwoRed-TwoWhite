## 📖 About This Project

This site serves as my Digital Flight Bag for Private Pilot Ground School. It is a curated collection of videos, notes, podcasts, and various online resources that I found useful.

While I strive for accuracy in documenting these aviation concepts, please note that this is a living document. Should you encounter any typos or technical inaccuracies, please feel free to contact me so I can keep this resource as precise as possible for the community
